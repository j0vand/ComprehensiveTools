欢迎使用**腰突康复训练计时器**！

## ⚡ 快速开始（3步）

### 1️⃣ 打开应用
```
双击文件：rehabTrainer.html
或在浏览器中打开此文件
```

### 2️⃣ 添加训练项
```
方式1（推荐）：点击"快速添加示例训练"
方式2：手动点击"添加训练项"
```

### 3️⃣ 开始训练
```
点击绿色的"开始训练"按钮
跟随语音提示完成训练
```

---

## 🔄 跨设备同步（新功能！）

### 在旧设备上：
```
1. 点击"导出数据"按钮（绿色）
2. 保存下载的JSON文件
3. 发送到新设备（微信/邮件/云盘）
```

### 在新设备上：
```
1. 打开应用
2. 点击"导入数据"按钮（蓝色）
3. 选择之前导出的JSON文件
4. 确认导入
5. 完成！所有训练计划都同步过来了
```

## 💡 使用建议

### 第一次使用
1. 先用示例训练熟悉功能
2. 理解两种训练模式的区别
3. 体验语音引导效果

### 日常使用
1. 每天固定时间训练
2. 跟随语音不用看屏幕
3. 定期导出备份数据

### 数据备份
1. 每周导出一次
2. 保存到云盘
3. 换设备时导入

## ⚠️ 重要提示

1. **医疗建议**：训练计划应由专业医生或康复师制定
2. **量力而行**：如感到疼痛或不适，立即停止
3. **循序渐进**：从简单动作开始，逐步增加难度
4. **坚持训练**：康复需要时间和耐心
