🚀 如何使用：

1. 【最简单】直接双击打开 rehabTrainer.html

2. 【推荐】使用浏览器打开：
   - Chrome/Edge: 右键 → 打开方式 → Chrome
   - Safari: 右键 → 打开方式 → Safari

3. 首次使用：
   ✓ 系统会自动创建"我的训练计划"
   ✓ 点击"快速添加示例训练"体验功能
   ✓ 点击"开始训练"立即开始

4. 自定义训练：
   ✓ 点击"添加训练项"
   ✓ 选择类型（持续时间/次数）
   ✓ 填写配置信息
   ✓ 保存后即可使用

✨ 核心功能：

✓ 双模式训练（持续时间型/次数型）
✓ TTS语音全程引导
✓ 精确计时和进度显示
✓ 数据自动保存
✓ 暂停/继续/跳过功能
✓ 完美适配手机屏幕

📱 推荐使用设备：

✓ 手机浏览器（最佳体验）
✓ 平板浏览器
✓ 桌面浏览器

🔊 语音说明：

- 首次使用需点击屏幕激活语音（iOS）
- 确保设备音量打开
- 支持调节语速和音量
