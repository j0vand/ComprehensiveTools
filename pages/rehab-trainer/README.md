# 腰突康复训练计时器

一个专为腰椎间盘突出康复训练设计的网页应用，支持语音引导、计时提醒，帮助用户更好地完成康复锻炼。

## 功能特点

### 🎯 核心功能
- **双模式训练**：支持持续时间型（如平板支撑30秒）和次数型（如小燕飞10次）
- **语音引导**：全程TTS语音提示，无需看屏幕也能跟着练
- **智能计时**：精确倒计时、组间休息自动管理
- **数据保存**：训练计划本地保存，下次打开继续使用
- **暂停/继续**：支持训练中暂停和恢复
- **导入导出**：支持数据导出和导入，轻松在不同设备间同步

### 🗣️ 语音提示
- 准备开始："准备开始训练，10秒后开始"
- 训练提示："平板支撑，坚持30秒" / "小燕飞，做10次，完成后点击按钮"
- 进度提醒："还剩10秒，坚持住"
- 组间休息："第1组完成，组间休息60秒"
- 完成祝贺："全部训练完成，做得很棒！"

## 使用方法

### 1. 创建训练计划
- 首次打开会自动创建"我的训练计划"
- 可以通过"新建计划"按钮创建多个计划（如"第一阶段"、"第二阶段"等）

### 2. 添加训练项
点击"添加训练项"按钮，配置训练内容：

**持续时间型示例（平板支撑）：**
- 名称：平板支撑
- 类型：持续时间型
- 坚持时间：30秒
- 组数：3组
- 组间休息：60秒
- 说明：保持身体成一条直线，收紧核心

**次数型示例（小燕飞）：**
- 名称：小燕飞
- 类型：次数型
- 每组次数：10次
- 组数：3组
- 组间休息：60秒
- 说明：俯卧，双臂双腿同时抬起

### 3. 开始训练
1. 点击"开始训练"按钮
2. 听到"准备开始训练，10秒后开始"
3. 倒计时3、2、1后正式开始
4. 跟随语音提示完成训练

### 4. 训练中操作

**持续时间型：**
- 暂停/继续：随时暂停训练
- 跳过：跳过当前训练项
- 结束：提前结束训练

**次数型：**
- 完成一组：按自己节奏做完后点击此按钮
- 跳过：跳过当前训练项
- 结束：提前结束训练

## 训练项示例

### 腰突康复常见训练

**第一阶段（急性期恢复）：**
1. 小燕飞：10次 × 3组，组间休息60秒
2. 桥式：15秒 × 3组，组间休息60秒
3. 平板支撑：20秒 × 3组，组间休息60秒

**第二阶段（功能恢复）：**

2. 桥式：30秒 × 3组，组间休息60秒
3. 平板支撑：30秒 × 3组，组间休息60秒
4. 侧桥：20秒 × 2组（每侧），组间休息60秒

**第三阶段（强化训练）：**
1. 小燕飞：20次 × 3组，组间休息60秒
2. 桥式：45秒 × 3组，组间休息60秒
3. 平板支撑：45秒 × 3组，组间休息60秒
4. 侧桥：30秒 × 3组（每侧），组间休息60秒
5. 死虫式：12次 × 3组，组间休息60秒

## 技术说明

### 浏览器支持
- Chrome/Edge（推荐）
- Safari（iOS/macOS）
- Firefox
- 其他支持Web Speech API的现代浏览器

### 语音功能
- 使用浏览器原生Web Speech API
- 支持中文语音合成
- iOS设备首次使用需要点击屏幕激活语音功能

### 数据存储
- 使用localStorage本地存储
- 数据保存在浏览器中，不会上传到服务器
- 清除浏览器数据会丢失训练计划

## 注意事项

1. **医疗建议**：训练计划应由专业医生或康复师制定
2. **量力而行**：如感到疼痛或不适，立即停止训练
3. **循序渐进**：从简单的动作开始，逐步增加难度
4. **坚持训练**：康复需要时间，请保持耐心
